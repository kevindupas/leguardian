================================================================================
                              API ENDPOINTS
================================================================================

7 Endpoints Integrated (All Real API Calls):

1. POST /api/devices/auth
   - Authenticates bracelet
   - Response: Bracelet data + ID

2. POST /api/devices/heartbeat
   - Sends location + battery
   - Called every 5 seconds
   - Contains: latitude, longitude, battery_level

3. POST /api/devices/button/arrived
   - Child arrived safely event
   - Battery: -1%

4. POST /api/devices/button/lost
   - Child lost event
   - Battery: -1%
   - Alert level: High

5. POST /api/devices/button/danger
   - Emergency button pressed
   - Battery: -2%
   - Alerts: Parent + Authorities

6. GET /api/devices/commands
   - Poll for pending commands
   - Called every 5 seconds (with heartbeat)
   - Returns: List of commands from parent responses

7. POST /api/devices/commands/{id}/ack
   - Acknowledge command execution
   - Called immediately after processing command
